select控件示例说明

HTMLCode:
<span>
<select name="days" id="days" onchange="alert('ok');">
    <option value="30">30日</option>
    <option value="60">60日</option>
    <option value="90">90日</option>
    <option value="">全部</option>
</select>
</span>

一个简单的实例

JSCode:
var b = new xui.widgets.Select('days',80);
效果：

使用add方法添加一个选项

b.add('wori','fuck');:添加一个值为'fuck'，文本为'wori'的选择项
也可以传入一个option对象：
b.add({text:"wori",value:'fuck'});

JSCode:
var b = new xui.widgets.Select('days',80);
b.add('wori','fuck');
效果：

select方法

选中某项:select(v,txt);
如果没有第二个参数，则按所有选项的值来查询，否则按文本来查询
如果有相同选择，则选中索引最小的项
【相关方法】selectByIndex(index):按索引值来选中
如：selectByIndex(0);选中第一项

JSCode:
var b = new xui.widgets.Select('days',80);
b.select('90');
效果：

remove方法

删除某项:remove(txt,v)
只传入第一个参数并且参数为number类型的时候为删除第几项，为其他类型的时候将查找文本为txt的那项删除
如果第二个参数存在，则查找值为txt的那项删除
[*值为0的BUG未修正]

JSCode:
var b = new xui.widgets.Select('days',80);
b.remove(2);
效果：

setModel方法

setModel(obj);
obj:一个配置参数对象
obj对象的key字段说明:
  obj.msg:一个初始化的提示。
  obj.suggest:当值为'true'[String]的时候将开启静态suggest功能
  obj.selectModel:当值为'false'[String]的时候将没有下拉按钮
  obj.initValue:设置一个初始值。
  obj.autoClose:开启/关闭“自动关闭”选项。默认关闭，开启：{autoClose:'true'}[String]

JSCode:
var b = new xui.widgets.Select('days',80);
b.setModel({msg:'请选择'});
效果：

静态suggest功能

setModel({suggest:'true'});
在选择输入框输入字段后如果存在匹配则筛选出相应的选项

JSCode:
var b = new xui.widgets.Select('days',80);
b.setModel({suggest:'true'});
效果：

其他方法说

1：获取当前选中的值：getValue();
2：重新渲染原SELECT节点：reLoad();
3：删除当前的所有选项渲染一个新的数据列表：load(list); list[option]:参数为一个optionList对象，如：[{'text':'xxx','value':'yyy'}...]
4：将一个新的数据列表插入当前选项列表中：addList(list);list[option]:参数为一个optionList对象
5：选中了某个值后触发的事件：change();自定义事件[*注意不要和原select节点的onchange冲突]
6：选择某项值的时候触发的事件：onselect();自定义事件 返回值为Boolean类型
7：点击此select控件下拉按钮操作的时候：click();自定义事件 返回值为Boolean类型
var b = new xui.widgets.Select('days',80);
b.click = function(){alert('click');return true;};
b.onselect = function(){alert('onselect');return true;};
b.change = function(){alert('选中值为：'+b.getValue());};
效果：
